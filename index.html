HTML

HTML (Hypertext Markup Language) predstavlja osnovni jezik koji se koristi za kreiranje i strukturiranje web stranica. Iako se cesto naziva "programskim jezikom", HTML je zapravo oznacivacki jezik koji omogucava definisanje strukture i sadrzaja web stranica.HTML koristi razlicite oznake (tagove) kako bi definisao razlicite elemente na stranici. Ove oznake obuhvataju sve, od naslova i paragrafa do slika i veza. HTML nije samostalan jezik za programiranje, vec se cesto kombinuje s drugim jezicima, poput CSS (Cascading Style Sheets) za stilizovanje i JavaScript za dinamicke funkcionalnosti.

CSS

Cascading Style Sheets (CSS) predstavlja kljucnu tehnologiju koja omogucava dizajniranje i stilizovanje veb stranica. Dok HTML (HyperText Markup Language) definise strukturu sadrzaja, CSS pruza sredstva za oblikovanje tih elemenata. Bez CSS-a, veb stranice bi bile jednostavne i osnovne, lisene estetskih i korisnickih iskustava koje danas ocekujemo.Ove tehnike cine vase veb stranice privlacnijim i funkcionalnijim, cime poboljsavaju korisnicko iskustvo. Ovo je samo pocetak, a dalje istrazivanje i praksa ce vam pomoci da postanete strucnjak u kreiranju modernih veb stranica.

JS

JavaScript je programski jezik koji se Cesto koristi za razvoj dinamiCkih web stranica. JavaScript se izvrsava direktno u web pregledniku korisnika, omogucavajuci interaktivnost i dinamicke promene na stranici. Osnovna uloga JavaScripta je poboljsati korisnicko iskustvo na webu. Moguce je manipulisati HTML-om, stilovima i cak komunicirati s web serverom koristeci asinhrone HTTP zahteve (AJAX). Time se postize brzi odziv i korisnici dozivljavaju fluidnije interakcije na stranici. Jedna od kljucnih karakteristika JavaScripta je njegova sposobnost dogadjajskog programiranja.
